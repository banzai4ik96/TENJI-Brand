## Правила использования ядра

1. Для каждой страницы: 1 основной кластер + 2-4 дополнительных.
2. Основной ключ — в title, h1, первом абзаце и meta description (без переспама).
3. В каждом тексте использовать естественные формулировки и синонимы.
4. При добавлении новой страницы сразу добавлять ее в sitemap.xml.
5. Обновлять этот файл минимум 1 раз в неделю.
